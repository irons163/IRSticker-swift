# IRStickerSwift

IRStickerSwift is a lightweight sticker view for iOS.

## Swift Package Manager

Add the package via Xcode:

1. File > Add Packages...
2. Enter the repository URL.
3. Add the `IRStickerSwift` library to your target.

Then import it where needed:

```swift
import IRStickerSwift
```

## CocoaPods

CocoaPods is no longer supported.
